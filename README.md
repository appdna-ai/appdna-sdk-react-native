# AppDNA SDK for React Native

The official React Native SDK for [AppDNA](https://appdna.ai) — the growth console for subscription apps.

## Installation

```bash
npm install github:appdna-ai/appdna-sdk-react-native#v1.0.0
```

For iOS, also run:

```bash
cd ios && pod install
```

## Quick Start

```typescript
import { AppDNA } from '@appdna/react-native-sdk';

AppDNA.configure('YOUR_API_KEY');
```
